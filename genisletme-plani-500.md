## Uygulama Notları

**Altyapı:** `scripts/generate-icons.ts` değişiklik gerektirmez, subdirectory desteği zaten mevcut.
Yeni kategori eklemek için `svgs/<kategori>/` klasörü oluşturup SVG'leri eklemek yeterli.

**SVG Standardı:**
```
viewBox="0 0 24 24"
fill="none"
stroke="currentColor"
stroke-width="2"
stroke-linecap="round"
stroke-linejoin="round"
```

**Her Faz Sonunda Doğrulama:**
```bash
npm run generate   # ikon sayısı / kategori sayısı doğru mu?
npm run build      # hatasız
npm test           # tüm testler geçmeli
```

**Etkilenen Dosyalar (Her Faz):**
- `svgs/<kategori>/*.svg` — yeni SVG'ler
- `apps/demo/src/data/iconRegistry.ts` — import + icons dizisi güncellenir
